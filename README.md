OUTPUT:

STRAIGHT FORWARD PROMPT:

A straightforward prompt is a clear, simple, and direct instruction or question given to an AI. It does not include unnecessary details, complex wording, or multiple confusing requests. The goal is to tell the AI exactly what you want in a simple way.

EXAMPLES:

“What is artificial intelligence?”

“List creative names for a coffee shop.”

TABULAR FORMAT PROMPT:

Tabular format prompting is when you ask an AI to organize information into a table instead of writing it in plain text. This makes the output structured, easy to read, and comparable.

EXAMPLES:

“Compare the following smartphones in a table: iPhone 14, Samsung Galaxy S23, Google Pixel 7. Columns: Brand, Model, Price, Battery Life, Camera Quality.”

“Create a weekly task table with columns: Task, Day, Priority, Status.”

PRECEDING QUESTION PROMPT:

The preceding question prompt technique is a method where you ask the AI one or more “preliminary questions” before the main question to guide its reasoning or gather context. This is slightly different from just giving instructions — here you use questions to set the stage, so the AI thinks step by step before answering your main question.

EXAMPLES:

“Write an engaging introduction for a blog about healthy eating for teenagers.”

“Calculate the total distance a car travels if it moves at 60 km/h for 2.5 hours.”

MISSING WORD PROMPT:

A missing word prompt is a type of prompt where you give the AI a sentence, paragraph, or text with one or more words removed, and ask it to fill in the blanks.

The sun rises in the ____.”

Water ____ at 100°C and ____ at 0°C.”
